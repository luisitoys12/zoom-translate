Features

Everything you need.
Nothing you pay for.

Built to replace paid solutions with a fully open, extensible alternative.

🎙️
Real-time transcription
Web Speech API runs entirely in your browser. Audio never leaves your device.
🌐
Live bidirectional translation
ES ↔ EN ↔ FR ↔ PT ↔ DE ↔ JA. Powered by Google Translate API or LibreTranslate.
👥
Shared sessions
Share a link so guests see your live subtitles without installing anything.
📹
Works on Zoom, Meet & Teams
Overlay subtitles on any browser-based meeting. No Zoom account needed.
📤
Export transcripts
Download your meeting as .txt, .srt subtitles, or .docx document.
🔒
Privacy first
Only text (never audio) is sent to translation APIs. Full control over your data.
🧩
Supabase backend
Meeting history, session sharing and realtime sync powered by open source Supabase.
🌙
Dark mode + i18n
Full Spanish and English UI. System theme auto-detection.

Quick Start

Up and running in 2 minutes.

No account required for basic use.

# 1. Clone the repo git clone https://github.com/luisitoys12/zoom-translate.git # 2. Open Chrome → chrome://extensions → Developer mode ON # 3. Load unpacked → select the extension/ folder # Optional: run the web dashboard locally cd web && pnpm install && pnpm dev

Get a free Google Translate API key at Google Cloud Console (500k characters/month free).
